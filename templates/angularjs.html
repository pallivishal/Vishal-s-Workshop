Angular.js
Linux Flavour: Debian

  1. Installing Angular

    1. apt-get install npm
    2. npm --version
    3. npm install -g @angular/cli
    4. ng new angular-web-component
    5. cd angular-web-component
    6. ng add @angular/elements
    7. ng generate component MyWebComponent
    8. ng serve --open

  2. Creating a Project

    We will create a project using Angular CLI

    1. Set up your enviroment.

      • Install the Angular CLI

        Setting up the local enviroment and workspace

        1. apt-get install npm
        2. npm install -g @angular/cli

      • Create a workspace and initial application

        To create a new workspace and initial starter app:

        1. Run the CLI command ng new and provide the name my-app, as shown

          ng new my-app

        2. The ng new command prompts you for information about features to include in the initial app. Accept the defaults by pressing the Enter or Return key.

    2. Run the application

      The Angular CLI includes a server, so that you can build and serve your app locally.

      1. Navigate to the workspace folder, such as my-app.

        cd my-app

      2. Run the following command:

        ng serve --open

        If, port 4200 is already in use, then try this below command:

        ng server --port port_number --open

    3. Make changes to the application


      • THE HERO EDITOR

        1. Create the heroes component

          Using the Angular CLI, generate a new component named heroes.

          ng generate component heroes

          The CLI creates a new folder, src/app/heroes/, and generates the three files of the HeroesComponent along with a test file.
