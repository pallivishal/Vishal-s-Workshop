    • int

      'int' is a data type in c language which is used to store integer value in variable.

                    int counter = 0 ;

      We can also increase an interger value of a variable without actually assigning a increased value to that variable.

                    counter = counter + 1;

      There are two other ways of writing the above code. One is:

                    counter += 1;

      And the other is:

                    counter++;

      The above two variation of writing the same increment code is known as syntactic sugar.
